# Motrakor-todolist

This is a simple To-Do-List command-line application written in Python. Users of the application will be able to perform the following tasks.
<li>Sign up and log in (login details are stored in a JSON file)</li>
<li>Create and edit a to-do-list item</li>
<li>View all to-do-list items</li>
<li>View to-do-list item details</li>
Title
Details
Priority (high, low, mid)
Status (completed, pending)
Owner
Updated date
Created date
<li>Mark a to-do-list item as completed</li>
